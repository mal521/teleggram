# teleggram

a cozy web app where hallabong picks a love letter for you based on your mood, delivered via gyelan eggs.

## what it does

- slide between **playful** and **warm** moods
- hallabong (a mandarin orange) follows the slider and picks an egg for you
- each gyelan egg holds an affirmation — tap one to reveal it
- share your love letter via a link that auto-plays the egg animation

## how to run

open `index.html` in a browser. no build step, no dependencies.

## tech

plain html/css/js. google fonts (quicksand). no frameworks.
